# Mumbai-Housing-Price-Predictor-
Want to buy a house in mumbai? Use this free tool to get an estimate of what could cost you.
Ensure the two files are in the same folder, then copy the path of the CSV file and paste it onto the python file's 12th line.
Ensure the following libraries are downloaded : 

Pandas: For data manipulation.
NumPy: For numerical operations.
Scikit-learn: For machine learning models.
Matplotlib or Seaborn (optional): For visualizing the model's performance.
Tkinter: For building the GUI.
